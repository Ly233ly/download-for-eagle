## 修复与回归

- 同页同类型候选使用响应 Digest/Content-MD5/强 ETag，或“长媒体路径 + 精确总字节数”建立通用强别名。
- 强别名只能唯一并入一个显式播放器组；歧义时不猜测。
- 组内备用 CDN 地址折叠为一个下载版本，保留真实标题、帧、时长和质量最完整的候选。
- 修复前 Node 回归稳定得到 2 组；实现后为 1 组、1 版本。
- 相同大小但不同路径的媒体仍为 2 组；响应指纹相同但路径不同的别名可正确归并。
