# `cat-catch` 功能对等矩阵

> 历史基线与退役记录：本矩阵证明 1.0.0 曾覆盖的行为，不表示 1.2.0 仍应装载全部能力。当前活动边界见决策 65–69、A145–A151 和 `.scratch/extension-boundary-1-2/PRD.md`。

- 基线：`xifangczy/cat-catch` 2.7.1，提交 `7a77612b3e2a01cedacae6e43eb88a89eee3034f`。
- 功能基线：下载中转站 `1.0.0`；方案 2 单界面与本机统一下载目标：`1.1.0`。
- 当前状态：仅捕获、归组、预览、筛选、配对、来源规则、增强发现与任务控制继续保留；录制、缓存代理、移动 UA、密钥、旧清单/下载/预览页、浏览器 FFmpeg、自动下载和外部目标均从 1.2.0 活动载荷退役。所有下载动作统一提交给本机软件。

证据索引：

- [`tests/test_upstream_parity.py`](../tests/test_upstream_parity.py) 现在验证固定 GPL 对应源码与通知仍在，并验证旧上游工具箱不进入活动扩展；功能正确性由扩展、媒体协调器和 API 回归分别覆盖。

## 1.2.0 退役清单
