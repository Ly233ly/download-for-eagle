# Domain Docs

本仓库是单一上下文项目。

## 开始工作前

- 阅读根目录 `CONTEXT.md`，使用其中定义的领域词汇。
- 阅读 `docs/DECISIONS.md` 中的现有边界。
- 涉及 `cat-catch` 集成时同时阅读 `docs/CAT_CATCH_MIGRATION_PLAN.md` 和 `docs/FEATURE_PARITY_MATRIX.md`。

当前架构决策统一记录在 `docs/DECISIONS.md`。如果实现与现有决策冲突，必须显式指出并更新该文档，不能静默覆盖旧决定；只有仓库未来正式引入独立 ADR 目录后才按 ADR 文件维护。
