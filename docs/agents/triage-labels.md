# Triage Labels

| 规范角色 | 本仓库状态值 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 等待维护者评估 |
| `needs-info` | `needs-info` | 等待补充信息 |
| `ready-for-agent` | `ready-for-agent` | 说明和验收完整，可由代理独立实现 |
| `ready-for-human` | `ready-for-human` | 需要人工决策或真实环境验收 |
| `wontfix` | `wontfix` | 决定不实施 |

状态写在本地任务文件的 `Status:` 行；不要为了同步状态创建同名 GitHub 标签。
