## What to build

把默认 popup/侧边栏改为内容级媒体库。使用页面标题、Open Graph、video poster、可安全加载的媒体元数据和受限首帧生成，让用户看到下载内容；同组内通过清晰度、音轨和字幕选择器形成明确下载方案。

## Acceptance criteria

- [x] 打开扩展首先看到媒体候选组，不需要再点击“弹出预览”才能识别内容。
- [x] 每组展示标题、站点、封面/首帧、时长、分辨率、帧率、编码、码率、大小和输出名中的可用字段。
- [x] 无法生成预览时显示 CORS、认证、编码或资源失效等具体原因，并保留可下载判断。
- [x] 直链和 HLS 可内嵌播放；DASH/分轨候选显示组合预览或明确的分轨预览状态。
- [x] 支持按类型、站点、大小、时长、清晰度、可预览性和状态筛选、排序、分页和去重。
- [x] 下载按钮旁明确展示将下载的流、合并方式、输出容器和预计大小。
- [x] 原始 URL、请求 ID 和调试字段默认折叠，键盘与屏幕阅读器可操作主要流程。
