# 验收标准

## A. 核心自动化

- **A01**：IDM 完成受支持的视频下载后，系统自动创建导入任务。
- **A02**：Eagle 正在运行时，视频自动出现在当前资源库的“全部”中。
- **A03**：没有来源网页时仍然导入，Eagle 来源字段保持为空。
- **A04**：捕获到可靠来源时，Eagle 来源字段等于视频所在网页地址，不使用视频直链或猜测网址。
- **A05**：Eagle 未运行时任务保持“等待 Eagle”，启动后无需重新下载即可补导入。
- **A06**：导入完成后，下载目录中的原视频仍存在且内容未改变。

## B. 网站规则与来源

- **A07**：用户可在 Chrome 中开启或关闭当前网站自动导入。
- **A08**：开启网站后，扩展识别到的下载动作会保存网页来源。
- **A09**：关闭网站后，扩展识别到的下载动作会生成跳过指令；无法被浏览器识别的 IDM 动作仍按“无来源直接导入”处理。
- **A10**：可配置多个网站并正确处理常见子域名。
- **A11**：来源不确定时不得猜测；导入记录明确显示“未记录来源”。
- **A12**：已导入且有 Eagle 项目编号的记录可以事后补写来源，不重复导入文件。

## C. 文件与重复保护

- **A13**：非视频文件不会导入，并留下“非视频忽略”原因。
- **A14**：至少支持 MP4、MOV、MKV、WebM、AVI、M4V、MPEG、MPG、TS、M2TS 和 WMV。
- **A15**：相同内容的视频再次下载时跳过；不同内容即使同名也不能误判。
- **A16**：中文名称、空格、括号和较长 Windows 路径可以正确处理。
- **A17**：文件不存在、为空或仍在增长时不得提前导入。
- **A18**：同一路径在短时间内被 IDM 重复通知时只保留一个活动任务。

## D. 记录、恢复与占用

- **A19**：右下角存在托盘图标；左键可打开记录，右键显示 Windows 原生菜单，记录窗口可区分成功、等待、失败和跳过。
- **A20**：失败任务可以手动重试，错误原因对普通用户可理解。
- **A21**：程序或电脑意外关闭后，未完成队列不会丢失。
- **A22**：新下载通过轻量信号立即唤醒处理；空闲时不高频轮询，窗口隐藏后降低界面刷新频率。
- **A23**：默认记录保留 90 天、最多 10,000 条，并可手动清理。
- **A24**：助手和 IDM 接收器不会弹出终端窗口。

## E. 安全与回滚

- **A25**：数据只保存在本机，不调用 Eagle 之外的外部服务。
- **A26**：不保存 Cookie、账号密码、浏览器请求头或网页正文。
- **A27**：本机接口只监听 `127.0.0.1`，Chrome 扩展使用配对码、扩展来源和随机令牌认证。
- **A28**：安装前记录 IDM 原设置；停用后可以恢复。
- **A29**：卸载不会删除下载视频或 Eagle 项目。
- **A30**：网站规则、完整来源和本地路径不会写入对外诊断文件。

## F. 分享与一键安装

- **A31**：发行 ZIP 自带运行环境，未安装 Python 的 Windows 10/11 电脑也可运行。
- **A32**：一键安装器把完整程序复制到用户本地应用目录，创建桌面与开始菜单入口并注册卸载项。
- **A33**：IDM 对应设置为空或旧助手路径时自动配置为接收器与 `[File]`；发现其他安全软件配置时不覆盖。
- **A34**：安装前 IDM 原值只备份一次；卸载时仅在当前值仍指向本助手时恢复，避免覆盖用户后续修改。
- **A35**：安装器生成的自动配对凭据每台电脑不同、七天过期、仅允许 Chrome 扩展 Origin，并在成功配对后删除。
- **A36**：安装器自动打开 Chrome 扩展页面和正确文件夹；由于浏览器安全限制，最终“加载已解压”由用户亲自确认。
- **A37**：文件尚未稳定时界面明确说明会自动重试，单次等待不超过约 3 秒且最多 20 次；用户无需手动操作，也不会无限循环。
- **A38**：已经成功导入的同一任务不得因手动重试竞态变为“重复跳过”；历史误报记录在升级后自动恢复为“导入成功”。
- **A39**：普通临时错误最多自动重试 12 次；Eagle 长期未启动最多低频等待约 1 小时，达到上限后停止并允许用户手动重新开始。
- **A40**：同一网页产生多条重复视频请求时只作为一个来源候选；不得因候选自身重复而漏写可靠来源。
- **A41**：正常运行时任务管理器不得把本助手显示为 Python；主程序、后台、托盘和扩展应显示“下载中转站”及自有图标。

每项验收至少保留自动测试、截图、任务记录或 Eagle 实际结果之一；没有验证的项目不得标记为完成。
