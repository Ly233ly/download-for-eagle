# 安全说明

请勿在 Issue、日志或截图中公开配对令牌、更新签名私钥、本机完整路径、下载记录或其他私人信息。

发现安全问题时，优先使用 GitHub 仓库的 **Security > Report a vulnerability** 私下报告；若该入口尚未开启，请先创建不包含漏洞细节的普通 Issue，请维护者开启私密沟通。

发布包不得包含 `data/`、`runtime/`、`release/`、本机数据库、配对文件或任何私钥。公开仓库只保存源码、文档、公开图标与构建说明。

## 媒体安全边界

- 捕获到的 Cookie、Authorization、Referer、Origin、User-Agent 和临时签名地址只允许为当前经过配对认证的本机下载任务最小化使用；页面显式请求头与浏览器观测请求头只按固定白名单合并。完整地址和请求头不写入扩展持久存储或数据库；数据库只保存移除查询参数的诊断地址。普通媒体上下文仅在任务内存存在。YouTube/通用页面 yt-dlp 解析是受控例外：Cookie 上限为 16 KiB，转换为任务专用 `临时/<planId>/*-cookies.txt`，其路径而非内容进入进程参数，并在解析成功、失败或取消的 `finally` 中立即删除。任务终止或软件重启后上下文失效，必须回来源页重新创建，不允许降级到浏览器下载。
- 当前页截图只用于缺少公开媒体封面的可见预览，保留在当前 popup 内存中，不进入扩展持久存储、助手数据库、日志、诊断或外部服务；通用内容图只接受 HTTP(S) 图片地址，拒绝脚本和其他 URL scheme。
- 扩展只提交媒体候选组、用户选择、允许的运行时请求头和导入选择；本机 API 校验配对令牌、扩展 Origin、任务 schema、body 大小和速率。旧组件路径、Chrome downloadId、组件完成/失败 API 与 DNR 临时规则已经删除。
- 普通直链、分离音视频、HLS/DASH、“下载并导入”和“仅下载”全部由安装包内固定 FFmpeg/ffprobe 执行。popup、预览页、直链页和解析页都提交相同认证计划；旧自动下载和捕获后自动外发已删除。新版界面不向浏览器 FFmpeg/WASM、在线处理页、Aria2、MQTT、send2local、m3u8dl 或自定义命令发送下载内容。
- 外部程序参数使用结构化模板和允许列表，不拼接到 shell；yt-dlp 只允许经过验证的公网 HTTP(S) 内容页面、固定格式表达式、固定 Deno 路径及临时 cookie 文件路径。本机、环回和字面内网 IP 页面在建计划时拒绝；敏感凭据不得出现在命令预览、错误或诊断导出中。
- Widevine、SAMPLE-AES 等 DRM 只检测、说明并阻断，不提供规避、密钥提取或解密能力。
- 自动清理仅适用于程序专用临时根目录内、带任务归属记录的中间文件；最终下载文件、原下载文件与用户手动指定文件永不自动删除。
- 诊断导出提供预览，并移除令牌、Cookie、完整本机路径以及 URL 查询参数。
- 网络代理只按任务作用于远程媒体输入；localhost、环回、非公网字面地址、Eagle 和助手 API 永远直连。手动设置只接受无认证的 HTTP/Mixed 主机端口，避免代理密码写入数据库、进程参数或诊断；诊断仅保留模式、来源和主机端口。
- cat-catch 2.7.1 固定源码、GPL-3.0、修改说明、项目 MIT 保留通知和完整构建材料随 1.0.0 对应源码提供。

自动安全回归覆盖 loopback/Origin/配对令牌、Firefox Origin、运行时请求头白名单、受保护直链、DRM 前置阻断、CR/LF 请求头注入、签名 URL 脱敏、任务取消、重启后上下文失效和只清理 owned-temp。浏览器内部扩展页仍需用户亲自加载和确认，安装器不绕过 Chrome 安全提示。
