下载中转站 1.2.11

安装方法
========

1. 先把整个 ZIP 解压到普通文件夹，不要直接在压缩包预览窗口里运行。
2. 双击“一键安装.exe”。
3. 点击“一键安装”。
4. 安装器会自动配置 IDM、创建“下载中转站”桌面快捷方式并启动软件。
5. Chrome 扩展管理页和扩展文件夹会自动打开：
   - 打开右上角“开发者模式”；
   - 点击“加载已解压的扩展程序”；
   - 选择已经打开的 chrome-extension 文件夹。

扩展加载后会自动安全配对，不需要输入六位配对码。

日常使用
========

- 点击扩展后，媒体页会显示标题、封面、类型、清晰度、音轨和大小；选择后可以“仅下载”或“下载并导入 Eagle”。
- B 站等音视频分离内容会显示为一个项目，由本机自动下载并合并，不重编码。
- 打开 Eagle 后，照常在 Chrome 中使用 IDM 下载视频。
- 下载中转站没开也没关系，下载完成时会自动在右下角启动。
- 没捕获到来源网页时仍会直接导入；捕获到时会把原网页写入 Eagle。
- 点击右下角下载中转站图标可以查看记录；右键可以打开网站规则或退出。
- 网络默认自动跟随 Windows 系统代理。Chrome 能打开网站但下载报 403 时，进入软件顶部“网络”；如果自动模式显示未检测到系统代理，选择“手动代理”，填写代理软件的 HTTP/Mixed 端口，例如 127.0.0.1:7890。

注意
====

- 支持 Windows 10/11；IDM/Eagle 用于原自动导入流程，媒体扩展支持 Chrome/Edge，并附 Firefox 独立清单。
- 安装包自带 Python 运行环境和 FFmpeg/ffprobe，不需要另外安装。
- DRM 内容会明确提示不支持；软件不会把媒体上传到第三方服务。
- 如果 IDM 原本已经配置了杀毒软件，安装器会保护原设置而不强行覆盖，并在完成页提示手动处理。
- Chrome 会提示“开发者模式扩展”，这是加载个人未上架扩展时的正常提示。
- 原下载文件不会被移动、删除或修改。

卸载
====

进入“Windows 设置 > 应用 > 已安装的应用”，找到“下载中转站”并卸载；也可以使用开始菜单里的“卸载助手”。卸载会恢复安装前的 IDM 设置，不会删除下载视频或 Eagle 里的项目。
